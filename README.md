# Yukari Uploader

I make this application for simple file sharing.

## Enviroment

- macOS Mojave 10.14.4(18E226)
- NodeJS v10.16.0

## Usage

```
node app.js
```

## Issue

### Background Image Change

Change `public/img/bg.jpg` to new Image.

or 

Fixed background url in `index.html`.

```
background: url("/img/bg.jpg");
```
